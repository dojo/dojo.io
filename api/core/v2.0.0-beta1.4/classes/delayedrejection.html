Class DelayedRejection

A Promise that will reject itself automatically after a time. Useful for combining with other promises in Promise.race.

  • Inherited from ExtensiblePromise.all

    • Defined in src/async/ExtensiblePromise.ts:63

    Return a ExtensiblePromise that resolves when all of the passed in objects have resolved. When used with a key/value pair, the returned promise's argument is a key/value pair of the original keys with their resolved values.

    example

    ExtensiblePromise.all({ one: 1, two: 2 }).then(results => console.log(results)); // { one: 1, two: 2 }
