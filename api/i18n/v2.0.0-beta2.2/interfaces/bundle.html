Properties

bundlePath

bundlePath: string
  • Defined in src/i18n.ts:24

The absolute module ID used by the loader to resolve paths to locale-specific paths. This MUST follow the format "{basePath}{separator}{filename}". For example, if the module ID for a bundle is "nls/common", the loader will look for locale-specific bundles at "nls/{locale}/common".

Optional locales

locales: string[]
  • Defined in src/i18n.ts:29

A list of supported locales. Any included locale MUST have an associated bundle.

messages

messages: T
  • Defined in src/i18n.ts:35

The map of default messages that will be used when locale-specific messages are unavailable. Note that any message key used in the i18n system MUST have a default specified here.
