Class ReadableStreamReader <T, T >

This class provides the interface for reading data from a stream. A reader can by acquired by calling {@link ReadableStream#getReader}. A ReadableStream can only have a single reader at any time. A reader can be released from the stream by calling ReadableStreamReader.releaseLock. If the stream still has data, a new reader can be acquired to read from the stream.
