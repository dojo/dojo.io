sandbox

  • sandbox(doc?: Document): Handle
    • Defined in src/mixins/Projector.ts:84

    Attach the project to a sandboxed document fragment that is not part of the DOM.

    When sandboxed, the Projector will run in a sync manner, where renders are completed within the same turn. The Projector creates a DocumentFragment which replaces any other root that has been set.

    Parameters

    • Optional doc: Document

      The Document to use, which defaults to the global document.
